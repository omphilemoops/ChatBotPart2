# Cybersecurity Awareness Chatbot

## Overview
The Cybersecurity Awareness Chatbot is a WPF-based desktop application developed using C# and XAML in Visual Studio. The chatbot is designed to educate users about cybersecurity threats and safe online practices through an interactive graphical user interface (GUI).

The application allows users to enter a username and interact with the chatbot while displaying user-related information such as the last seen time. The chatbot responds to cybersecurity-related keywords and topics including phishing, passwords, fraud, malware, hacking, online scams, and digital safety.

---

## Features
- User-friendly WPF graphical interface
- Username input and user interaction
- Displays user last seen time
- Responds to cybersecurity-related keywords
- Cybersecurity awareness and educational tips
- Interactive chatbot conversation system
- Keyword recognition for security topics
- Modern desktop application design

---

## Cybersecurity Topics Covered
The chatbot can provide information and guidance on topics such as:

- Phishing attacks
- Password security
- Online fraud and scams
- Malware and viruses
- Cyberbullying
- Social engineering
- Safe browsing practices
- Data privacy
- Two-factor authentication
- Account protection

---

## Technologies Used
- C#
- WPF (Windows Presentation Foundation)
- XAML
- Visual Studio 2022
- .NET Framework

---

## How to Run the Project
1. Open the project in Visual Studio 2022.
2. Build the solution.
3. Run the application.
4. Enter a username to begin chatting with the chatbot.

---

## Purpose of the Project
The purpose of this project is to create awareness about cybersecurity by providing users with quick information and guidance regarding common cyber threats and online safety practices.
